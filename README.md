# Product and Category CRUD Search Project

This is a simple project

## Installation

- git clone `https://github.com/Yunandarmoe/product-category.git`

- Open project folder

- Copy .env file

- Create database same with .env database name

- php artisan migrate

## Finally run

- php artisan serve
